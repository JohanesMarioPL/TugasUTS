Apa itu HTML?
HTML atau HyperText Markup Language merupakan sebuah bahasa markup standar dalam pembuatan suatu web. HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web.
Yang perlu diperhatikan ketika mulai belajar HTML
1. Extensi File HTML
Setiap file HTML harus berekstensi
.html
Jika tidak menggunakan ekstensi ini, maka ia tidak akan bisa dibaca oleh web browser.
2. Nama Khusus untuk homepage
Jika kamu ingin membuat halaman untuk homepage, maka sebaiknya gunakan nama
index.html
.Karena ia akan otomatis dibuka saat website dikunjungi.

    3.Mengenal Tag, Element, Atribut dan Value

  • dengan kurung siku (<...>), lalu di dalamnya berisi nama tag dan kadang juga ditambahkan
  • dengan atribut.
    • Element (Elemen)
  • Elemen dibentuk dari tag pembuka, isi tag, dan tag penutup. Kadang juga ditambahkan
  • beberapa atribut.
    • Attribute (Atribut)
  • Atribut adalah kata khusus yang berada di dalam tag pembuka. Atribut juga disebut sebagai
  • modifier yang akan menentukan perilaku dari elemen. setiap Tag memiliki atribut yang
  • berbeda-beda sehingga kita perlu mempelajarinya dengan sabar.
